# **App Name**: Academia Access

## Core Features:

- Secure Google Login: Users can securely log in using their institutional Google accounts, enforcing domain restrictions to ensure authorized access.
- Visitor Check-in Form: A user-friendly interface for visitors to input their college department and reason for visit, with automatic timestamping upon entry. This data is saved to Firebase.
- Role-Based Access Control: Distinguishes between Admin and Visitor roles, granting appropriate access and visibility to different application features.
- Real-time Admin Dashboard: Admins access a dashboard displaying real-time visitor statistics, filterable by day, week, and month to monitor usage trends from Firebase.
- User Search & Management: Admins can efficiently search for specific users by name and manage their accounts, including the ability to block or unblock users. Data updates are performed via Firebase.
- AI-Powered Usage Insights: An LLM-powered tool generates summary reports for administrators, highlighting busiest hours and most common visit reasons to identify valuable usage patterns based on historical data from Firebase.

## Style Guidelines:

- Primary color: A sophisticated, muted blue-purple (#474766) to convey professionalism and academic integrity.
- Background color: A very subtle, light blue-purple (#EFEFF3), offering a clean and unobtrusive canvas for content.
- Accent color: A clear, vibrant sky blue (#429ED1) used for interactive elements and key highlights to provide visual contrast and guidance.
- Body and headline font: 'Inter' (sans-serif) for its modern, neutral, and highly legible appearance, suitable for professional and data-rich interfaces.
- Utilize a set of clean, simple, and intuitive line icons to maintain a professional aesthetic and enhance usability.
- Implement a clean, well-structured layout with clear information hierarchy to ensure data is easily digestible and navigation is intuitive, adapting gracefully to different screen sizes.
- Subtle and functional animations to provide feedback on user interactions and smooth transitions between dashboard views, enhancing the overall user experience without being distracting.
